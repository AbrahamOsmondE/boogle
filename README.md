# Boogle

Mobile web app made to practice and play the board game boggle. Deployed with AWS ECS/Fargate with ALB and all that stuff.

https://booooooogle.com
# Frontend Stack

- React
- Redux
- AWS SDK

# Backend Stack

- Express
- Websocket.io
- redis
- PGSql

# DevOps/CICD

- AWS (Lambda, S3, Route 53, Cloudfront)
- AWS CDK
- Github Actions
- Docker/Docker compose
- Devcontainer
